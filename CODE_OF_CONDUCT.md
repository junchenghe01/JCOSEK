# Contributor Covenant Code of Conduct

We value respectful, inclusive interactions. All contributors are expected to follow the Contributor Covenant 2.0.

Key points:
- Be considerate, respectful, and collaborative.
- Avoid harassment, personal attacks, or discriminatory remarks.
- Focus on the problem, not the person.

If you experience or witness unacceptable behavior, contact the maintainers at: hejuncheng0@gmail.com
